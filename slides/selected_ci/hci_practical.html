Bottlenecks (Optional)

  1. Matrix diagonalization (Davidson's Algorithm)
    • Large sparse matrix vector multiplication
    • Non-standard storage of matrix makes it tougher to optimize
    • Memory/speed trade-off in matrix storage
  2. Evaluating and appending CI Hamiltonian
    • Typically 106 x 106 or larger (stored sparsely)
    • Not easy to use standard containers because the matrix changes dimensions every iteration
  3. Searching for new/"important" determinants
    • Look at connected determinants for each determinant in the active space
