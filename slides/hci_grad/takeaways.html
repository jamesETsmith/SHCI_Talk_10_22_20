Takeaways

  • Final geometries and energies converge smoothly to the exact answers
  • We can run "cheap" geometry optimizations and if we want accurate energies we can run a "tight" HCI calculation
  • Even with the ability to treat larger active spaces, we still find systems with orbitals "rotating" in/out of the active space
